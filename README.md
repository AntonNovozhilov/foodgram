Foodgram - приложение чтобы делиться рецептами. 
Вы можете:
- Создавать свои рецепты.
- Смотреть рецепты других пользователей.
- Добавлять/удалять рецепты в избранное.
- Скачавать список покупок.

Скачать репозиторий
```

```
